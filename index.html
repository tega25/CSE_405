About Snort

Snort is an open source network indtrusion detection system. It is a packet sniffer that monitors network traffic in real time, it scans each packet that is being transmitted to determine if the packet is safe. If the packet is viewed as unsafe, Snort could send alerts to email or to another server to notify that their is a mailicious packet in the system. If the packet is safe, Snort could still be configured to notify if the packet validates a set rules that Snort has to offer. Snort is able to detect DOS, buffer overflow, stealth port scan attacks.

Research

After installing Snort I configured a basic set up of Snort. I had monitor traffic that was being transmitted on my web server. I used the rules that were in the .rules file that was an example. I was able to get snort to alert me when I started to ping my VM that I setup Snort to monitor. I learned that Snort is an extremely in depth IDS and when setup correctly it can save you from various attacks.
